# 💳 Credit Card Fraud Detection System

## 📌 Project Overview
This project is a Machine Learning model that detects fraudulent credit card transactions using Logistic Regression. It helps identify whether a transaction is normal or fraudulent based on transaction patterns.

---

## 🎯 Objective
To build a simple and effective fraud detection system that can classify transactions as:
- Normal Transaction (0)
- Fraud Transaction (1)

---

## 📊 Dataset
A public credit card fraud dataset containing transaction details like:
- Amount
- Time
- Features V1 to V28 (anonymized)
- Class label (0 = normal, 1 = fraud)

---

## ⚙️ Tech Stack
- Python
- Pandas

- Scikit-learn

- Seaborn
- Google Colab

---

## 🧠 Machine Learning Model
- Logistic Regression

---

## 🔄 Workflow
1. Load dataset
2. Data preprocessing
3. Train-test split
4. Feature scaling
5. Train Logistic Regression model
6. Make predictions
7. Evaluate model performance

---

## 📈 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

---

## 🚨 Output Example
- Fraud Transaction → 🚨 Fraud Detected
- Normal Transaction → ✅ Safe Transaction
